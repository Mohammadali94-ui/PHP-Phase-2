# COMP1006 Phase 2 Project

## Overview
This project is a PHP web application that allows users to register, log in, and manage their account. It includes session authentication, CRUD functionality, and file uploads.

## Features
- User registration with validation
- Login system with password hashing
- Session-based authentication
- Protected pages (only accessible when logged in)
- File upload functionality
- Edit and delete account options

https://php-phase2.page.gd/

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## How to Run Locally
1. Install XAMPP
2. Place project folder inside `htdocs`
3. Start Apache and MySQL
4. Import the SQL file into phpMyAdmin
5. Update `db.php` with your database credentials
6. Open `http://localhost/your-folder-name`
